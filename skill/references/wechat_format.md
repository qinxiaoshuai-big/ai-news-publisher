# 微信公众号 HTML 格式规范

## 核心限制
- 不支持外部 CSS：所有样式必须用内联 style="" 属性
- 不支持 iframe, video, script, embed 等标签
- 图片仅支持 jpg/png/gif，建议宽度 900px 以内
- 字体颜色建议用 #333 #666 #1a1a1a 等标准色

## 推荐样式参数

| 元素 | 字号 | 行高 | 颜色 | 备注 |
|------|------|------|------|------|
| 正文 | 15px | 1.75 | #333 | 标准阅读体验 |
| H2 | 18px | 1.5 | #1a1a1a | 加粗 |
| H3 | 16px | 1.5 | #333 | 加粗 |
| 引用 | 14px | 1.6 | #666 | 左边框+灰底 |
| 代码 | 13px | 1.6 | #abb2bf | 深色背景 |
| 链接 | 继承 | 继承 | #576b95 | 微信默认链接色 |

## 图片要求
- 格式：JPG / PNG / GIF
- 上传方式：cgi-bin/media/uploadimg → 返回 URL
- 封面图：cgi-bin/material/add_material → 返回 media_id

## 草稿箱 API 限制
- 标题不超过 64 字
- 摘要不超过 120 字（不传自动截取正文前 54 字）
- access_token 有效期 7200 秒，需缓存复用

## 常见错误码
| errcode | 说明 |
|---------|------|
| 40001 | access_token 过期或无效 |
| 40164 | IP 不在白名单 |
| 40007 | media_id 无效 |
| 40125 | AppSecret 错误 |
